# Kit Schema

A kit is a JSON file describing a collection of agent skills and configs. Generated by `clawclawgo pack`.

## Structure

```json
{
  "name": "My Kit",
  "description": "What this kit does",
  "version": "1.0.0",
  "exportedAt": "2026-03-14T18:00:00.000Z",
  "compatibility": ["claude-code", "cursor", "openclaw"],
  "detectedFiles": ["SKILL.md", "CLAUDE.md", ".cursorrules"],
  "skills": [ ... ],
  "configs": [ ... ],
  "scan": { ... },
  "repository": "https://github.com/user/repo"
}
```

## Top-Level Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | yes | Kit name (defaults to directory name) |
| `description` | string | no | What this kit does |
| `version` | string | no | Semver version |
| `exportedAt` | string | no | ISO 8601 timestamp of when `pack` ran |
| `compatibility` | string[] | yes | Which agents this works with |
| `detectedFiles` | string[] | no | Agent config files found in the directory |
| `skills` | Skill[] | yes | List of skills found |
| `configs` | Config[] | no | Agent-specific config files |
| `scan` | ScanResult | yes | Baked-in security scan results |
| `repository` | string | no | Git remote URL (auto-detected) |

## Skill

A skill detected from a `SKILL.md` file following the [Agent Skills](https://agentskills.io) standard.

```json
{
  "name": "voice-assistant",
  "description": "Process voice commands and respond with TTS",
  "path": "skills/voice-assistant",
  "compatibility": "openclaw, cursor, windsurf",
  "license": "MIT",
  "allowedTools": ["exec", "read", "write"]
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | yes | Skill name (from SKILL.md frontmatter) |
| `description` | string | no | What this skill does |
| `path` | string | yes | Relative path to the skill directory |
| `compatibility` | string | no | Agent compatibility (from frontmatter) |
| `license` | string | no | License identifier |
| `allowedTools` | string[] | no | Tools this skill uses |

## Config

An agent-specific config file (not a SKILL.md).

```json
{
  "file": "CLAUDE.md",
  "agent": "claude-code",
  "preview": "# Claude Code Instructions\n\nWhen reviewing code..."
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `file` | string | yes | Filename |
| `agent` | string | yes | Which agent uses this file |
| `preview` | string | no | First ~500 chars of content |

### Detected Config Files

| File | Agent |
|------|-------|
| `CLAUDE.md` | claude-code |
| `.cursorrules` | cursor |
| `.windsurfrules` | windsurf |
| `AGENTS.md` | openclaw |
| `openclaw.json` | openclaw |
| `codex.json` | codex |
| `.clinerules` | cline |
| `.aider.conf.yml` | aider |
| `.continue/config.json` | continue |

## Scan Result

Security scan results baked into the kit at pack time.

```json
{
  "score": 95,
  "findings": [
    {
      "severity": "low",
      "type": "network",
      "message": "External API call in setup.sh",
      "file": "skills/voice/scripts/setup.sh",
      "line": 12
    }
  ],
  "timestamp": "2026-03-14T18:00:00.000Z"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `score` | number | Trust score 0-100 |
| `findings` | Finding[] | List of security findings |
| `timestamp` | string | When the scan ran |

### Finding

| Field | Type | Description |
|-------|------|-------------|
| `severity` | string | `block`, `warn`, or `low` |
| `type` | string | Category: `injection`, `exfiltration`, `credentials`, `pii`, `dangerous`, `network` |
| `message` | string | Human-readable description |
| `file` | string | File where the issue was found |
| `line` | number | Line number |

## Compatibility IDs

Use these IDs in the `compatibility` array:

`claude-code` · `cursor` · `windsurf` · `openclaw` · `codex` · `cline` · `aider` · `continue` · `github-copilot` · `vs-code` · `gemini-cli` · `roo-code` · `goose` · `openhands` · `opencode` · `amp` · `junie` · `pi` · `factory` · `trae`

See [AGENT-COMPATIBILITY.md](https://github.com/bolander72/clawclawgo/blob/main/AGENT-COMPATIBILITY.md) for the full list of 30+ agents.
